Phase 0 — Research

Validator rewards for XRPL, built by the community.

XNODE is an open initiative to prototype a DAO-governed, sustainable rewards layer for XRPL validators — using transparent telemetry, a DAO‑managed budget, and on-ledger payouts.

Open-Source XRPL-first Telemetry-driven

Why XNODE?

XRPL validators secure the network but have no native rewards. XNODE proposes a fair, opt-in incentive layer that leaves consensus untouched.

How it Works

Watchers collect uptime & agreement; a Reward Engine scores epochs; the DAO sets budget B; payouts stream on-ledger.

Built for Sustainability

Diversified treasury inflows (LP fees, grants, bond fees) and adaptive budgeting reduce sell pressure and extend runway.

System Architecture

Validators → Watchers → Reward Engine → DAO Treasury → XRPL Payouts

Telemetry

Independent watcher agents collect uptime, agreement %, and latency. Attestations cross-check integrity.

DAO Treasury

Multi-sig to start; later token-holder voting. Epoch budgets align incentives with treasury health.

DeFi Hooks

Optional MPT/TEA validator bonds, AMM liquidity, and streaming payouts via payment channels.
